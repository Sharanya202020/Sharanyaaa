My Projects

A collection of academic and engineering projects focused on IoT, Automation, Embedded Systems, and Digital Design.

IoT-Based Cold Storage Monitoring System

Developed an intelligent monitoring system to continuously track temperature and humidity inside cold storage facilities. The system provides real-time environmental monitoring and generates alerts when conditions exceed safe limits, helping to reduce spoilage and improve storage efficiency.

Technologies: IoT, Sensors, ESP32, Monitoring Systems

Automatic Cat Sensor Door Opening System

Designed a smart pet access solution that automatically opens a door when a cat is detected nearby. The system utilizes sensors and automated control mechanisms to provide secure, convenient, and hands-free access for pets.

Technologies: Sensors, Automation, Embedded Systems

Automated Toll Booth System

Developed an automated toll collection model capable of detecting vehicles and controlling gate operations without manual intervention. The project demonstrates concepts used in intelligent transportation systems and smart infrastructure.

Technologies: Automation, Sensors, Traffic Systems

Two-Way & Three-Way Traffic Signal Logic Using Vivado

Designed and simulated traffic signal controllers using Verilog HDL in Vivado. Implemented finite state machine (FSM) logic for both two-way and three-way intersections, ensuring proper signal timing, coordination, and safe traffic flow.

Technologies: Verilog HDL, Vivado, FPGA Design, FSM
