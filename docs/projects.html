IoT-Based Cold Storage Monitoring System

Developed an IoT-based monitoring system to continuously track temperature and humidity inside cold storage facilities. The system provides real-time monitoring and alerts when conditions move beyond safe limits, helping reduce spoilage and maintain product quality.

Automatic Cat Sensor Door Opening System

Designed a smart pet access system that automatically opens a door when a cat is detected nearby. The project uses sensors and automated control mechanisms to provide convenient and secure access for pets without human intervention.

Developed an automated toll collection model capable of detecting vehicles and controlling gate operations without manual intervention. The system improves traffic flow and demonstrates automation concepts used in intelligent transportation systems.

Designed and simulated traffic signal controllers using Verilog HDL in Vivado. Implemented finite state machine logic for both two-way and three-way intersections with proper timing sequences, state transitions, and signal coordination.
